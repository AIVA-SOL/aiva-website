🔧 第一步：安装 Python

1下载 Python 安装包

点击下面的链接，下载 Python 官方安装包（Windows 64位）：

⬇ 下载 Python 3.12.3 (官方) ⬇ 下载 Python 3.12.3 (华为镜像，国内加速)
💡 推荐使用华为镜像，国内下载速度更快
2运行安装程序（重要：勾选 PATH）

双击下载的 .exe 文件，安装时 务必勾选 以下选项：

☑  Add Python 3.12 to PATH   ← 这个必须勾！！！
☑  Install for all users
然后点击 "Install Now"
⚠ 如果忘记勾选 "Add Python to PATH"，安装完后运行 启动.bat 会提示找不到Python。解决方法：重新运行安装包 → 选 "Modify" → 勾选 PATH 选项。
3验证 Python 安装成功

按 Win + R，输入 cmd，回车打开命令提示符，然后输入：

python --version
✅ 正确输出示例：Python 3.12.3

如果显示 'python' 不是内部或外部命令，说明 PATH 没有配置，需要重新安装并勾选 PATH 选项。

❓ 常见问题 FAQ

Q1: 打开页面没有数据，一直显示"加载中"？

原因：后端还在筛选中（首次运行需要3-5分钟）

解决：耐心等待，查看后端命令行窗口是否有输出。如果5分钟后仍无数据，检查网络是否能访问 api.binance.com（可能需要VPN）。

Q2: 顶部显示"连接断开"（红色点）？

原因：后端服务未启动，或后端崩溃

解决：1) 确认后端命令行窗口没有关闭；2) 检查是否有报错信息；3) 重新运行 启动.bat

Q3: 安装依赖时报错 "timeout" 或 "connection error"？

解决：手动安装，在 cmd 中执行：

cd /d D:\workbuddy智能AI\虚拟币\backend
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/

如果还是超时，换用腾讯云镜像：

pip install -r requirements.txt -i https://mirrors.cloud.tencent.com/pypi/simple/
Q4: 访问 api.binance.com 超时（在中国大陆）？

原因：币安和 Gate.io 的 API 在中国大陆需要代理才能访问

解决方案一：使用 VPN 或代理，确保能访问外网

解决方案二（无需VPN）：在 backend/.env 中配置代理：

HTTP_PROXY=http://127.0.0.1:7890
HTTPS_PROXY=http://127.0.0.1:7890

（7890 改为你本地代理的端口号）

Q5: 推特数据显示"无账号"或情绪分为50？

原因：Nitter 公共镜像访问受限，或目标币种没有对应推特账号

说明：这不影响主要功能。推特情绪只占综合评分的10%权重，技术指标+回测才是核心。

Q6: 想接入 API Key 获得更高 API 限制怎么办？

编辑 backend/.env 文件，填入对应交易所的 API Key：

BINANCE_API_KEY=你的币安API Key
BINANCE_SECRET_KEY=你的币安Secret Key
GATE_API_KEY=你的Gate API Key
GATE_SECRET_KEY=你的Gate Secret Key

（不填也可以正常使用公开行情数据，只是有请求频率限制）

Q7: 如何修改筛选参数（比如改成价格 < 5 USDT）？

两种方式：

1. 前端实时调整：直接在页面的"价格上限"输入框改值，然后点"立即筛选"

2. 修改默认值：编辑 backend/src/config.py 中的 PRICE_MAX = 1.0 改为 5.0
